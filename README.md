Ring Crafting Database
======================

RUNNING THE APP:
Double-click "Open App.bat"

That's it. No installation needed.

The app opens in its own window (no browser tabs, no address bar)
using Microsoft Edge, which is built into Windows 10/11.

If Edge isn't available it will try Chrome, then fall back to
your default browser.

YOUR DATA:
All data saves automatically and persists between sessions.
Your rings, sale prices, and sales orders are always restored.

FEATURES:
  Database  — manage ring recipes, ingredient costs, sale prices
  Full Recipe — see complete material breakdown for any ring
  Sales     — build orders, mark customer-supplied materials,
               view profit and Total Cost Owed
